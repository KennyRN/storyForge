# Theme Licences

This project bundles colour palettes derived from the following third-party, open-source themes. Attribution and licence details for each are below. Full licence text is given once, at the end of this document.

Palette values are reproduced from each theme's canonical source. Where a theme publishes multiple named variants, only a representative light and/or dark variant is bundled.
